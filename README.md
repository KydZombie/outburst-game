# Outburst
A card-fighter game where emotions effect the game.

## Build Requirements
- .NET 8+
- Godot Mono 4.6

## How to Build & Run
### Terminal Version
- Clone repo
- `cd Terminal`
- `dotnet build` to build (Executable will be in `./bin/Debug/net8.0`) OR `dotnet run` to run immediately

### Godot Version
- Clone repo
- Open project in Godot
- Run project
